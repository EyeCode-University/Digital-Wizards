# 💎 Тема:Кошельки (интерфейс как в банковском приложении)

Example: Создайте React-приложение, которое отображает банковские кошельки, похожие на карточки счетов в приложении Сбербанка.

Каждый кошелек должен быть отдельным компонентом, а данные передаваться через props.

На странице должно отображаться несколько кошельков пользователя.

Пример использования компонента:

<pre>
    <WalletCard
  title="Основной счет"
  balance="125 340"
  currency="₽"
  cardNumber="4821"
  color="#21A038"
/>
</pre>
