## Суть задания

Создай React-приложение, которое работает с **твоим собственным API**. Регистрируешься на [MockAPI.io](https://mockapi.io), создаёшь данные, подключаешь их к React и выводишь красиво на экран.

| 🖥️ **Серверы** | Мониторинг серверов, статусы систем, метрики |

## 📋 Пошаговый план

### 1️⃣ Создай MockAPI

Зайди на https://mockapi.io и создай endpoint с **минимум 5 полями**.

**Пример для кофейни:**
```json
{
  "id": "1",
  "name": "Капучино",
  "description": "Эспрессо с молочной пеной",
  "price": 250,
  "imageUrl": "https://example.com/cappuccino.jpg"
}
```

**Пример для игр:**
```json
{
  "id": "1",
  "title": "Elden Ring",
  "genre": "Action RPG",
  "rating": 9.5,
  "cover": "https://example.com/elden.jpg"
}
```

> ⚠️ Добавь минимум **8-10 записей** с реальными данными

---

### 2️⃣ Напиши React-приложение

| Функционал | Описание |
|------------|----------|
| 📡 **Получение данных** | Используй `useEffect` для запроса к API |
| ⏳ **Loader** | Покажи анимацию загрузки во время запроса |
| ❌ **Обработка ошибок** | Выведи сообщение, если запрос упал |
| 🎴 **Карточки** | Отобрази данные в виде красивых карточек |
| 📱 **Адаптивность** | Сделай так, чтобы работало на телефонах |

---

### 3️⃣ Suspense

```
┌─────────────┐
│   LOADING   │ → показываем loader
└─────────────┘
┌─────────────┐
│   SUCCESS   │ → показываем данные
└─────────────┘
```
