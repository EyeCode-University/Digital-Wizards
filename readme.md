# 🧾 Задание: Форма перевода и история транзакций (React)

Представь, что у тебя маленький интерфейс онлайн‑банка. Нужно сделать форму перевода на карту и список истории переводов, который живёт в `state` приложения.

---

## Что должно получиться

У тебя есть один экран с двумя зонами:

1. **Форма перевода**  
   Пользователь заполняет:
   - номер карты получателя;
   - сумму перевода;
   - (по желанию) комментарий к переводу.

   При отправке формы создаётся объект транзакции и добавляется в состояние.

2. **История переводов**  
   Ниже формы отображается список всех совершённых переводов из `state`:  
   каждый элемент истории показывает номер карты (можно маскировать), сумму и время/дату создания записи.

---

## Ключевые требования

Форма:

```tsx
// примерная идея, а не готовый код
const [cardNumber, setCardNumber] = useState('');
const [amount, setAmount] = useState('');
const [comment, setComment] = useState('');
```

При отправке:

- предотвращаешь перезагрузку страницы (`event.preventDefault()`);  
- создаёшь новый объект транзакции, например:

```ts
{
  id: Date.now(),
  cardNumber: '** ** **** 1234',
  amount: 5000,
  comment: 'За кофе',
  createdAt: new Date()
}
```

- добавляешь его в массив `transactions` в `state`:

```tsx
const [transactions, setTransactions] = useState([]);
setTransactions(prev => [newTransaction, ...prev]);
```

История:

- выводишь `transactions` через `map`, самый свежий перевод сверху;
- оформляешь список аккуратно: сумма, номер карты, время/дата;
- можешь маскировать карту (оставлять только последние 4 цифры).

---

## Детали для красоты

- Добавь простую валидацию: не отправлять форму, если номер карты или сумма пустые/некорректные.
- После успешной отправки очищай поля формы.
- Минимально стилизуй интерфейс, чтобы форма и история визуально были разделены.

---

## Результат

В итоге у тебя есть:

- рабочая форма перевода, которая «пушит» транзакции в `state`;
- живая история переводов, которая обновляется после каждого отправления формы;
- весь функционал реализован на React c использованием `useState`.
