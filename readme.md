## 💡 Суть задания

Создай React-приложение, которое работает с **твоим собственным API**. Регистрируешься на [MockAPI.io](https://mockapi.io), создаёшь данные, подключаешь их к React и выводишь красиво на экран.

| Тема | Примеры данных |
|------|----------------|
| ☕ **Кофе** | Каталог напитков, кофейни, рецепты |
| 🖥️ **Серверы** | Мониторинг серверов, статусы систем |

## 📋 Пошаговый план

### 1️⃣ Создай MockAPI

Зайди на https://mockapi.io и создай endpoint с **минимум 5 полями**.

**Пример для кофейни:**
```json
{
  "id": "1",
  "name": "Капучино",
  "description": "Эспрессо с молочной пеной",
  "price": 250,
  "imageUrl": "https://example.com/cappuccino.jpg"
}
```

**Пример для игр:**
```json
{
  "id": "1",
  "title": "Elden Ring",
  "genre": "Action RPG",
  "rating": 9.5,
  "cover": "https://example.com/elden.jpg"
}
```

> ⚠️ Добавь минимум **8-10 записей** с реальными данными

---

### 2️⃣ Напиши React-приложение

Твоё приложение должно уметь:

- 📡 Получать данные через `useEffect`
- ⏳ Показывать loader во время загрузки
- ❌ Обрабатывать ошибки
- 🎴 Отображать данные в виде карточек
- 📱 Работать на мобильных устройствах

---

### 3️⃣ Обработай состояния

Приложение должно корректно работать в **трёх состояниях**:

```
LOADING → показываем loader
ERROR → показываем ошибку
SUCCESS → показываем данные
```
