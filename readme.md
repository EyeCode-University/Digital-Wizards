## 📝 Задание

Создайте React-приложение, которое получает данные из вашего собственного REST API на платформе [MockAPI.io](https://mockapi.io/) и отображает их в виде карточек.

## 🎨 Выберите тему проекта

- ☕ **Кофе** — каталог напитков или кофеен
- 🖥️ **Серверы** — мониторинг инфраструктуры
- 🎮 **Игры** — библиотека игр или персонажей
- 👕 **Одежда** — каталог товаров

---

## ✅ Технические требования

### 1. MockAPI Endpoint
- Минимум **5 полей** в модели данных
- Минимум **8-10 записей**

**Пример структуры:**
```json
{
  "id": "1",
  "name": "Капучино",
  "price": 250,
  "imageUrl": "...",
  "category": "Горячие"
}
```

### 2. React компонент
- Использовать `useEffect` для запроса данных
- Реализовать 3 состояния: **loading**, **error**, **success**
- Адаптивная вёрстка (mobile-friendly)

### 3. Обязательные элементы UI
- ⏳ Loader при загрузке
- ❌ Обработка ошибок
- 🎴 Карточки с данными
