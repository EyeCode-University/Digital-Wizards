# 💎 Тема: Отображение списка подключённых пользователей в реальном времени

Example: Необходимо доработать существующий мессенджер Pulse Messenger, добавив блок со списком пользователей, которые сейчас находятся онлайн.
Список должен обновляться в реальном времени при подключении и отключении пользователей.

Основная цель — научиться работать с WebSocket-событиями и синхронизацией состояния клиентов.
