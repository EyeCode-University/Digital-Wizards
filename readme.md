## Описание

Вам нужно создать React-приложение, которое работает с REST API. Данные будут храниться на MockAPI.io — это бесплатный сервис для создания тестовых API.

---

## Что делать

**Шаг 1.** Зарегистрируйтесь на https://mockapi.io и создайте новый проект

**Шаг 2.** Выберите тему для вашего приложения:
- Каталог кофе/напитков
- Мониторинг серверов
- Библиотека игр
- Магазин одежды

**Шаг 3.** Создайте endpoint с минимум 5 полями (например: id, название, описание, цена, картинка)

**Шаг 4.** Добавьте 8-10 записей с реальными данными

**Шаг 5.** Напишите React-приложение, которое:
- Получает данные через useEffect
- Показывает loader во время загрузки
- Обрабатывает ошибки
- Отображает данные в виде карточек
- Адаптируется под мобильные экраны

---

## Пример данных

Для кофейни:
```json
{
  "id": "1",
  "name": "Капучино",
  "price": 250,
  "imageUrl": "https://...",
  "category": "Горячие напитки"
}
```

Для игр:
```json
{
  "id": "1",
  "title": "Elden Ring",
  "genre": "RPG",
  "rating": 9.5,
  "imageUrl": "https://...",
  "year": 2022
}
```

---

## Требования к коду

1. Использовать useState для данных, loading и error
2. Использовать useEffect для запроса
3. Обработать все три состояния: загрузка, ошибка, успех
4. Чистый читаемый код

---

## Что сдавать

Загрузите проект на GitHub (без папки node_modules) или отправьте ZIP-архив.

В файле README.md напишите:
- Какую тему выбрали
- Ссылку на ваш MockAPI
- Как запустить проект (npm install && npm start)
- Приложите скриншот
